# Proyecto_Logica_de_programacion
Este proyecto corresponde al desarrollo del juego  "Adivina el número", donde el usuario piensa un número entre 1 y 50 y la computadora intenta adivinarlo en un máximo de 5 intentos.
# Funcionalidades
Mostrar instrucciones al usuario.
Validar respuestas ingresadas.
Calcular intentos mediante búsqueda binaria.
Limitar el juego a 5 intentos.
Mostrar si la computadora ganó o perdió.
Permitir jugar nuevamente.
